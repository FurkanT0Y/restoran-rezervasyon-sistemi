# restoran-rezervasyon-sistemi
Python ile geliştirilen konsol tabanlı sistem; dinamik menü yönetimi, akıllı masa doluluk kontrolü ve iç içe döngülerle çoklu müşteri sipariş takibi sağlar. try-except ile kullanıcı hatalarını önler. Siparişleri ve tarih damgalı mali raporları siparisler.txt/gun_sonu.txt dosyalarında kalıcı olarak depolar.
